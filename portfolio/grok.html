Grok Learning

2015 – 2017: Design and development work on the Grok Learning website completed while working as an engagement engineer .

Visit the website

Description

Grok Learning is an online platform with a mission to teach coding and help teachers deliver coding in their classroom. The Grok Learning website can be roughly split into:

  1. Customer pages - seen by all logged out users
  2. Learning pages - seen by logged in users
  3. Learning interface - delivering the learning material
  4. Admin pages - for authoring courses, managing customers etc.

I was heavily involved in the (re)design and development of the following parts of the Grok Learning website:

  • Marketing webpages (1)
  • Marketing emails (1)
  • User login and registration (1)
  • Bulk student subscription and payment (2)
  • Course launchpad (2)
  • Achievements (2)
  • Teacher dashboard design (2)
  • Learning interface (3)
  • Interactive tutor chat design (3)
  • Web design competitions (3)
  • BBC micro:bit course (3)
  • Python turtle Year 7 Maths & Computing course(3)
  • Python competiton problem authoring (3)
  • Custom CRM - customer messaging & segmentation, integrated interaction tracking, marketing campaign tools (4)
