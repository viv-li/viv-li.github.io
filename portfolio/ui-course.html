UI Design Course

Jan–May 2021: I created a user interface design course for Year 10-12 students across Australia to fulfill the Digital Technologies curriculum as an educational consultant.

Description

Grok Learning wanted to create an entirely new UI design course for Year 10-12 students across Australia. Project goals were:

  • curriculum alignment - students learn to "...design, prototype, and critically evaluate user interfaces using user stories"
  • course progression - to have it build on top of an existing web design course and teach new web design concepts
  • content engagement - to make the content more engaging by reducing written complexity and leveraging new interactive capabilities of the Grok platform

With my unique combination of learning design skills and professional UI and web design experience, I helped to shape and draft the course, which was then completed on the platform internally.

  1. Course design
    • Course shaping - selecting HTML/CSS concepts to follow on from beginners
    • Curriculum alignment - selecting UI design concepts
    • Delivery design - leveraging new platform features, how to teach UI design within platform
    • Web design - interactive slide example and problem theme brainstorming and selection

  2. Course creation
    • Outlining teaching progression slide by slide
    • Drafting 8 modules of slide content
    • Creation of slide example and problem designs
